# RustWeek 2025 Website

The website is built with [Astro](https://astro.build/).

In Astro, you can author content in [GitHub Flavored Markdown](https://github.github.com/gfm/).

## Setup project 

```
yarn install
```

## Run in development

```
yarn run dev
```

To check the build succeeds:

```
yarn run build
```
